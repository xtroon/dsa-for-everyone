## 🛠️ VS Code Setup & Execution

Since this repository is managed locally inside VS Code, here are some helpful tips for running the C++ files:

### Prerequisites

1. **Compiler**: Ensure you have a C++ compiler installed (e.g., `g++` via MinGW for Windows).
2. **Extensions**: Install the **C/C++** extension by Microsoft and **Code Runner** for easier execution.

### Compiling and Running

To manually run any file (e.g., `duplicate.cpp` inside the `array` folder) from the VS Code terminal:

```bash
# Navigate to the folder (e.g., array)
cd array

# Compile the C++ file
g++ duplicate.cpp -o duplicate

# Run the executable
.\\duplicate.exe
```

_Alternatively, click the **Run** button at the top-right corner if you have Code Runner installed._
